# Prima Facie

**At first sight: read faster and better with Optimal Recognition Point (ORP) highlighting.**

Prima Facie is a modern, fast, and elegant speed reading application built with React. It uses Optimal Recognition Point (ORP) technology to highlight the ideal focal point of each word, allowing you to double or triple your reading speed without sacrificing comprehension.

## Features

- **Document Support**: Drag & drop PDF, EPUB, and TXT files directly into your browser.
- **ORP Highlighting**: Automatically calculates and highlights the optimal recognition point for each word.
- **Chunking**: Read multiple words at a time with customizable chunk sizes.
- **Audio Metronome**: Optional rhythmic audio cues to keep your pace.
- **Text Preview**: Follow along with the full document text on the side, fully formatted for EPUBs.
- **Bookmarks**: Save your exact reading position and jump back instantly.
- **Keyboard Navigation**: Keep your hands on the keyboard with quick shortcuts (Space, Arrow keys, B for bookmarks).
- **Customizable**: Dark/Light mode, font sizing, and Words Per Minute (WPM) controls.

## Development

First, install dependencies and run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) with your browser to see the result.
